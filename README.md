# Smart Study Assistant

## 📌 Overview

Smart Study Assistant is a simple AI-based web application that helps students identify whether a subject is **weak or strong** based on their study habits.

The system uses a **machine learning model** to analyze:

* Study hours
* Completion rate
* Score

and provides instant feedback.

---

## 🚀 Features

* 📊 Predicts weak or strong subject
* 🤖 Machine Learning model (Random Forest)
* 🌐 Simple web interface using Flask
* 📈 Displays model accuracy
* 🎨 Clean UI

---

## 🛠️ Technologies Used

* Python
* Flask
* Scikit-learn
* Pandas
* HTML & CSS

---

## ▶️ How to Run Locally

1. Clone the repository:

```
git clone https://github.com/Vienna3007/smart-study-assistant.git
```

2. Navigate to the folder:

```
cd smart-study-assistant
```

3. Install dependencies:

```
pip install -r requirements.txt
```

4. Run the application:

```
python app.py
```

5. Open in browser:

```
http://127.0.0.1:5000
```
